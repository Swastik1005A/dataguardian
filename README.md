# 📊 DataGuardian

**DataGuardian** is a Python library for automated dataset diagnostics and data quality analysis.

It helps data scientists identify critical issues in datasets before model training—ensuring better data quality, faster debugging, and more reliable machine learning pipelines.

---

## 🚀 Features

- 🔍 Missing value detection  
- 🔁 Duplicate row detection  
- ⚖️ Target imbalance analysis  
- 📊 Correlation & multicollinearity detection  
- 🧪 Data leakage detection  
- 📈 Outlier detection (IQR method)  
- 🧠 Smart feature warnings (ID columns, useless features)  
- 📊 Data Health Score  
- 📉 Visualization support for EDA  
- 📄 HTML report generation
