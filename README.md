# 📊 DataGuardian

**DataGuardian** is a Python library for automated dataset diagnostics and data quality analysis.

It helps data scientists quickly identify issues in datasets before model training—saving time and improving model performance.

---

## 🚀 Features

- 🔍 Missing value detection  
- 🔁 Duplicate row detection  
- ⚖️ Target imbalance analysis  
- 📊 Correlation & multicollinearity detection  
- 🧪 Data leakage detection  
- 📈 Outlier detection (IQR method)  
- 🧠 Smart feature warnings (ID columns, problematic features)  
- 📊 Data Health Score  
- 📉 Visualization support for EDA  
- 📄 HTML report generation
